My Reflections

Technical Toolbox Skills I Learned

Bootstrap: Although I completed the lab to understand this framework, I chose to use media queries to create a responsive webpage in the technical documentation project. It was a learning curve understanding Bootstrap and it is a framework I would have to understand to be a better web designer.I chose to include a Bootstrap navbar as well as Bootstrap cards for this portfolio as opposed to a media query and Flexbox. Here is a cheat sheet for Bootstrap by Hackerthemes

Flexbox: I used this application in at least two projects. CSS Tricks has an illustrated chart on how Flexbox works as well as one for Grid.The chart helped explain how Flexbox works. In my recent project, I paired Flexbox with media queries to create responsive navigation.

GitHub: I found GitHub/Git to be the most "programmy" task I ever had to complete. It was a topic that I didn't know what I was getting myself into. I was given the impression that it was as simple as uploading a chosen project onto a webcloud for other web developers. Instead, I realized it was everything I described times ten. When I was working on the technical web page, I used a lab from the previous web development class as a test to upload. It took a really long time to understand what I am working with. Despite these difficulties, I intend to use GitHub for future projects because it is expected out of a front-end developer. As an update, I discovered a way to upload to repositories without using the Git version control system. GitHub has a drag and drop system that makes it easier for those who are not ready to use Git for creating and uploading to repositories.

My Journey in Web Development

Before taking COSW 20 and COSW10, I tried to teach myself how to code. I wanted to learn how to design beautiful web pages without relying on Wix, Weebly or similar webbsite building platform. So far, it's been a long road, and I want to say that I came a long way since I began learning HTML and CSS before taking these classes. I learned to think a little less like a creative, and a lot more like a programmer. It is not to say that I stopped being creative, because I still try to find creative ways to solve problems. Instead, I learned more about rules and structure. I intend to become a front-end web developer and use my skills in illustration, graphic design and web design to help me in this journey. I would also love to design graphics and web suitable icons using my skills in illustration. I will look back with amazement at the skills that I accomplished as I am now. I also intend to build on what I continue to learn, even if it means revisiting projects and reworking them.
